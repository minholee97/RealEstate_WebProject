# 🏡 Happy House

> 2022.05.13 ~ 2022.05.26 (팀 프로젝트)

- 원하는 지역의 부동산 매물 거래 정보를 조회할 수 있는 웹 서비스입니다.
- 지역 내의 매물 정보와 상권 정보를 조회할 수 있습니다.


## 1️⃣ 사용 기술
- JAVA 8
- Spring boot 2.7.0
- MySQL 8.0.27
- MyBatis
- JSP
